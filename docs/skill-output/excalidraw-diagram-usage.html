회로 다이어그램으로

복잡한 PMIC 시스템 요구사항을 말로 설명하면, /devx:excalidraw-diagram 스킬이 SPMI 버스·Vref 분배·ADC mux 경로까지 한 장에 담은 편집 가능한 다이어그램을 생성합니다.

6
PMIC 칩 (Main 1 + Sub 5)
16ch
ADC mux 입력
10
NTC 채널
SPMI
Main ↔ Sub 통신
1 · 실행한 명령

스킬 호출

$ /visuals-excalidraw-diagram docs/skill-output/example.txt

입력 파일 example.txt에 담긴 자연어 요구사항을 그대로 다이어그램으로 변환했습니다.

2 · 입력 프롬프트

무엇을 그려달라고 했나

“PMIC 시스템 회로 다이어그램을 만들어줘”라는 요청에 담긴 구조적 요구사항입니다.

🟢
Main PMIC ×1DSM ADC + SAR ADC, External 8ch, NTC 10ch 포함
🔵
Sub PMIC ×5각각 SAR ADC 1개씩 내장
🟣
SAR ADC + Vref기준 전압을 칩 바깥에서 공급받도록 표현
🟠
16ch Mux 입력ADC 입력 채널을 mux로 16개 표현
🔴
NTC 10ch온도 센서 채널 라인
⚡
External 8ch외부 소스 채널, 1 input으로 합류
세부 지시사항

디테일까지 반영

통신선

Main 과 Sub 의 통신선은 SPMI 로 명시 표기

ADC 모양

ADC 블록은 네모가 아니라 사다리꼴 ADC 기호 모양으로

NTC 라인

3개 라인만 그리고 중간에 ... 로 생략 표시

EX 채널

1 input 으로 들어가지만 일단 16개로 펼쳐서 표현

3 · 생성된 다이어그램

PMIC 시스템 아키텍처

1개의 Main PMIC + 5개의 Sub PMIC 구조를 SPMI 버스와 Vref 분배 경로까지 한 장에 표현했습니다. 아래는 결과물의 구조를 재현한 미리보기입니다.

pmic-system.excalidraw — excalidraw.com 에서 열어 자유롭게 편집할 수 있습니다.

사용 방법

3단계로 끝

1

스킬 호출

대화 중 언제든 스킬을 부르거나 입력 파일을 인자로 전달합니다.

/devx:excalidraw-diagram example.txt
2

다이어그램 설명

구조·연결·표기 규칙을 자연어로 구체적으로 묘사합니다.

Main PMIC 1개 + Sub 5개, SPMI 통신, ADC는 사다리꼴, mux 16ch, NTC 10ch, Vref는 칩 외부에서 공급 …
3

결과 확인 · 편집

생성된 파일을 excalidraw.com에서 열어 확인하고 다듬습니다.

# 생성 pmic-system.excalidraw # 열기 → excalidraw.com
활용 팁

더 좋은 결과를 위해

구조를 숫자로"칩 6개", "채널 16개"처럼 개수를 명시하면 레이아웃이 정확해집니다.
기호 모양 지정"ADC는 네모 말고 ADC 모양으로"처럼 도형 형태를 직접 요청할 수 있습니다.
생략 표시 요청반복 라인은 ... 로 줄여 그리도록 지시해 가독성을 높입니다.
반복 편집생성된 .excalidraw 파일을 excalidraw.com에서 자유롭게 수정하세요.
