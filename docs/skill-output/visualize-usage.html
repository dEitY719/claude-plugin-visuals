visualize 스킬은 마크다운(.md) 문서를 입력으로 받아, 별도 의존성 없이 브라우저에서 바로 열리는 자체 완결형 HTML(.html) 산출물을 생성합니다.

마크다운 문서
.md
평범한 텍스트 입력
/visuals-visualize
변환 스킬
내용을 시각 자료로
HTML 산출물
.html
단일 파일, 오프라인 OK
1

실행한 명령

범용 형식

/visuals-visualize <마크다운-문서-경로>

이번 예시

/visuals-visualize plugins/visuals/skills/visualize/SKILL.md
2

입력 — 마크다운 문서

  • 입력 파일: plugins/visuals/skills/visualize/SKILL.md
  • visualize 스킬의 정의가 담긴 평범한 마크다운 문서입니다.
  • 어떤 마크다운 문서든 입력으로 넣을 수 있습니다 — 위 파일은 하나의 예시일 뿐입니다.
3

결과 — HTML 산출물

위 명령을 실행하면, 입력 마크다운에 대응하는 자체 완결형 HTML output이 생성됩니다.

docs/skill-guides/visualize.html
